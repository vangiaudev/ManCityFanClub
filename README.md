## Available Scripts

In the project directory, you install node_module:
### `npm install --legacy-peer-deps`

After, you can run:
### `npm start`
or
### `yarn start`
Runs the app in the development mode.<br>

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
